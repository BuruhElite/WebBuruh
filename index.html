JAMINAN KEAMANAN MODAL:
Sistem hanya menggunakan koneksi API untuk eksekusi trade. Dana Anda tetap berada di wallet Binance. **Dilarang keras** mengaktifkan izin "Enable Withdrawals" pada pengaturan API Key Anda.

CARA CENTANG FUTURES DI BINANCE:
1. Buka akun Binance > API Management.
2. Klik 'Edit Restrictions' pada Key yang dibuat.
3. Centang kotak **'Enable Futures'**.
4. Jika tidak bisa dicentang, pastikan Anda sudah membuka akun Futures di menu Wallet Binance terlebih dahulu.
